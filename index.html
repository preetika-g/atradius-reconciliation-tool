User Guide
How to use the Atradius Reconciliation Tool — step by step.
Overview

This tool reconciles amounts collected by Atradius from retailers against open checkouts transferred to Atradius for recovery. Collections are matched to checkouts by Retailer ID, applied oldest due date first, and a checkout is marked paid when the collected amount reaches the match threshold (default 80%).

Monthly workflow
1

Upload collections

Go to Collections. Export the Atradius invoices sheet as CSV from the Google Drive folder and upload it. Rows are appended — you can upload multiple months and they accumulate.

2

Upload checkouts

Go to Checkouts. Export the Checkouts to Atradius sheet as CSV and upload it. Duplicate checkout IDs are automatically skipped so it is safe to re-upload the full file each month.

3

Check ID corrections

Go to ID Corrections. If Atradius uses a wrong Retailer ID in their file, add a correction here. The pre-loaded correction maps 3392 → 339266 (MAÏKAPO NATURAL). Add any new ones as you discover them.

4

Run the reconciliation

Go to Run & Results. Select the period pill for the current month (or "All periods" for a cumulative view). Keep the threshold at 80% unless instructed otherwise. Click Run reconciliation.

5

Review Gaps & Flags

Go to Gaps & Flags. Check the orange panel first — any collections with no retailer reference must be queried with Atradius. Then review unmatched collections (usually an ID mismatch — fix in ID Corrections) and open checkouts awaiting payment.

6

Export reconciled checkouts & update the tracker

Back in Run & Results, click Export reconciled CSV — this exports only the paid/reconciled checkouts. Open the reconciled checkouts tracker (link in Data Sources) and paste in the new rows.

7

Save your data

Click Save to browser in the top header. Your collections, checkouts and corrections are saved locally in your browser for next session. Click Restore saved next time you open the tool.

Status definitions
Paid in full100% of checkout amount collected.
Paid ≥ thresholdCollected ≥ threshold (e.g. 80%). Checkout reconciled.
PartialSome collected but below threshold. Remains open.
OpenNo collection received for this retailer.
